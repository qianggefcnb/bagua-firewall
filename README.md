# 🧱 八卦防火墙

基于阴阳八卦理论的智能防火墙系统

## 功能

- ☯️ 阴 (被动): 感知被攻击
- ⚡ 阳 (主动): 发现外部威胁  
- 🤖 AI检测: 集成学习异常检测
- 🔄 阴阳转换: 动态防御模式

## 模块

- `bagua_final.py` - 最终版
- `perception_ai_optimized.py` - AI优化感知
- `yin_yang_mode.py` - 阴阳转换

## 使用

```bash
python3 bagua_final.py
```

## 技术栈

- Python
- Scapy
- Scikit-learn
- Network Security
